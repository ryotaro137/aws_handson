# Text to Slide & 画像プロンプト生成 ワークフロー

このワークフローは、指定されたテキスト（例：今日の勉強記録の特定トピック）をMarp形式のスライドマークダウンに変換し、さらにNano Banana Proで画像生成するためのプロンプトマークダウンを一括生成します。

## ステップ 1: テキストからMarpスライドの生成

ユーザーからの指示や指定されたテキストを基に、`write_to_file` ツールを使用してMarp形式のマークダウンファイルを作成します。  
以下の点に注意して作成してください。

- ファイルの先頭に以下のフロントマターを含めること
  ```yaml
  ---
  marp: true
  theme: default
  paginate: true
  header: "AWS CloudOps 学習記録" # 必要に応じて変更
  ---
  ```
- 各スライドは `---` で区切る
- 箇条書きなどを活用し、簡潔で視覚的にわかりやすい内容にする
- （オプション）ユーザーから出力先ディレクトリの指定がない場合は `til` フォルダ等に保存する

## ステップ 2: スライドの分割とプロンプトファイルの生成

ステップ1で作成したMarpファイルを入力として、各スライドごとにNano Banana Pro用のプロンプトファイルを生成するPythonスクリプトを実行します。

// turbo
```bash
python .agents/scripts/generate_slide_prompts.py <ステップ1で作成したファイルのパス> <出力先ディレクトリのパス>
```

**仕様解説:**
- スクリプトは `---` を基準にスライドを分割します。
- 分割された各スライドのテキストを含むプロンプトファイルが `<元ファイル名>_prompt_01.md` のような連番で生成されます。
- `出力先ディレクトリのパス` は、指定がなければ元ファイルと同じディレクトリ内の `prompts` フォルダ等を指定してください。

## ステップ 3: 完了報告
生成されたMarpスライドと、画像生成プロンプトファイルの出力先をユーザーに報告して完了です。
